# Captures de témoignages WhatsApp

Déposez ici vos captures d'écran de témoignages clients (format `.jpg`, `.png` ou `.webp`).

## Recommandations

- **Format** : portrait (proportions d'un écran de téléphone, ~9:19).
- **Largeur conseillée** : 800 à 1080 px de large.
- **Nommage** : `whatsapp-1.jpg`, `whatsapp-2.jpg`, etc.
- **Confidentialité** : floutez les numéros de téléphone et données personnelles si nécessaire.

## Activation

Après avoir déposé une image, ajoutez son entrée dans :

```
src/data/testimonials.ts
```

La galerie premium (mockups de téléphone + lightbox + zoom + swipe) s'affichera automatiquement.
Tant que la liste est vide, des emplacements élégants « à venir » sont affichés à la place.
